# Portfolio

This is my personal portfolio website, showcasing some of my data and finance projects. You can visit it live at [alanvourch.com](https://alanvourch.com).

## What's inside

The site includes a filterable project gallery (filter by skill like Python, SQL, Power BI, Tableau), an about section, and a contact form. Projects cover areas like financial modeling, data analysis, machine learning, and business intelligence.

## Built with

- HTML, CSS, and vanilla JavaScript
- Tailwind CSS (via CDN)
- Hosted on GitHub Pages with a custom domain
